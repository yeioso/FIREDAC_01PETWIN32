IMPRIMIR FACTURA

Esta es una caja de dialogo ó pantalla que mostrará las facturas pendientes por imprimir, ofrece las siguientes opciones:

  • Modificar: Esta opción permite modificar información antes de de imprimirse totalmente (Puede adicionarse, anularse, eliminar ó modificar algun producto para ser tenido en cuenta al momento de facturarse).
  • Calcular: Ofrece una calculadora básica para efecto de devolución al momento del pago
  • Vista Preliminar: Muestra por pantalla la factura que se va a imprimir.
  • Precuenta: Imprime una factura para ser evaluada o aceptada por el cliente.
  • Imprimir: Imprime una factura que se le entregará al cliente.
  • Cuenta por Cobrar: Esta opción se encarga de llevar la factura a la cuentas por cobrar.

 

Modificar: Modifica algun registro de la factura, en este caso sería las operaciones sobre el total como lo es las propinas, descuentos, et; cuando vaya a modificar algun registro le pedirá el usuario y la contraseña, para efectos de log de actividades en cambios, y por ultimo mostrará una pantalla donde se puede cambiar el valor.

Para anular, adicionar, modificar o eliminar algun producto solo debe  presionar el icono donde esta la libreta  del recuadro rojo seleccionado, o si lo que desea es modificar los otros valores solo presiona la libreta y le solicitará la información que usted desee modificar, tambien puede modificar la forma de pago, el cliente y el numero de personas en la mesa.
