PROVEEDOR

Esta opción almacena la información básica de un proveedor y ofrece los siguientes campos:

  • Código: Almacena el número de identificación del proveedor como lo es el número de identificación tributaria, la cédula de ciudadanía, etc.
  • Nombre: Almacena el nombre ó la descripción del proveedor.
  • Dirección: Almacena la dirección del proveedor.
  • Teléfono 1: Almacena el teléfono del proveedor.
  • Teléfono 2: Almacena el teléfono del proveedor.
  • Subir Fotografía: Almacena la fotografía del proveedor.
