REGISTRAR APLICACION

Esta opción se encarga de registrar la aplicación para que opere el programa al 100%.

Si el programa no está registrado, solo podrá almacenar en cada archivo 10 registos, ademas al momento de realizar la impresión solo le mostrará un preview, sin permitirle enviar la información directo a la impresora.

El programa le mostrará el nombre de la maquina, el numero serial, solo deberá el código de autenticación que es proporcionado por el personal técnico del programa.
