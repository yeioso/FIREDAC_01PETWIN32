DATOS BASICOS

Esta opcion ofrece las opciones de menu para la ejecución del programa:

  • Zona:Almacena las diferentes zonas físicas que operará la atención de los clientes.
  • Grupo de Productos:Almacena los diferentes tipos de productos y/o servicios que se ofrecerán a los clientes.
  • Producto: Almacena los productos que se estará disponible para los clientes.
  • Adición: Almacena los diferentes productos que esta compuesto un producto, por ejemplo un coctail esta compuesto de varios licores.
