ATENDER LA MESA

Para administrar una mesa solo deberá presionar con el mouse en la imagen de la mesa.

Usted tambien puede mover la mesa teniendo la tecla control (Ctrl) y ubicandolo con el mouse

 

Aparecerá una caja de dialogo para llenar una información básica

Como se puede ver muestra los siguientes campos:

Estado de la Mesa: Ofrece dejar la mesa disponible, la mesa ocupada o la mesa reservada

Zona: Muestra el codigo de la zona, en la parte inferior aparece el nombre ó descripción de la zona.

Mesa: Muestra el codigo de la mesa.

Estacion: Muestra el nombre del computador

Personal: Muestra el nombre de la persona encargada de la mesa, esta puede deifnirse al momento de crear la zona o se puede cambiar en esta pantalla.

Nro. Personas: Este campo almacenará el número de personas que estará en la mesa.

Cliente: Muestra la identificación del cliente, se ofrece la opción de la busqueda, en caso que no exista, desde aca se puede crearlo.

El tipo de cliente: Se selecciona el tipo de cliente, debe tener en cuenta que si el cliente en el campo porcentaje tiene algun valor, primara sobre el porcentaje establecido en el tipo de cliente.

A la vez la pantalla puede ofrecer realizar operaciones entre las mesas

 

Puede pasar una mesa a otra, puede unir las varias mesas a una o solo modificar la información acerca del nombre de la mesa.
