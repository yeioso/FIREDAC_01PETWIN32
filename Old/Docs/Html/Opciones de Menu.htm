OPCIONES DE MENU

El sistema spiderPOS ofrece las siguientes opciones de menu para el manejo de si mismo.

  • Archivos: Ingreso de los datos básicos.
  • Movimientos: Son las diferentes operaciones que se puedean realizar para ejecuar el programa como son las compras, el punto de venta y el cierre de la caja.
  • Consultas: Ofrece las diferentes consultas personalizadas por el usaurio del programa.
  • Utilidades: Son las utilidades que ofrece el programa como lo son registrar la aplicación, generador de consultas, diseñador de informes y el cambio de contraseña.
  • Ayudas: Ofrece las ayudas del sistema.
