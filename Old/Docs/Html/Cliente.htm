CLIENTE

Esta opción almacena la información básica de un cliente y ofrece los siguientes campos:

  • Código: Almacena el número de identificación del cliente como lo es el número de identificación tributaria, la cédula de ciudadanía, etc.
  • Nombre: Almacena el nombre ó la descripción del cliente.
  • Dirección: Almacena la dirección del cliente.
  • EMail: Almacena el correo electrónico del cliente.
  • Teléfono 1: Almacena el teléfono del cliente.
  • Teléfono 2: Almacena el teléfono del cliente.
  • Subir Fotografía: Almacena la fotografía del cliente.
  • Descuento: Almacena el porcentaje del descuento en la venta por producto, si tiene algún valor primará ante la configuración del tipo de cliente.
