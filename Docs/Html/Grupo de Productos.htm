GRUPO DE PRODUCTOS

En esta opción almacena los diferentes tipo de productos que se ofrecerán a los clientes y tiene los siguientes campos:

  • Código: Almacena el código del grupo de productos.
  • Nombre: Almacena el nombre ó descripción de los grupos de productos.
  • Gráfico: Almacena el gráfico ó la imagen del grupo de productos.
  • Impresora: Almacena la impresora de salida para efectos de preparacion de algun producto.
  • Tipo de Proceso: Especial para aquellas empresas que al momento de realizar un pedido, inmediatamente realiza la impresión a una determinada impresora para efectos de preparación de productos de manera inmediata.
  • Mostrar grupo en el punto de venta: Este campo habilitará el grupo para ser seleccionado en el punto de venta
