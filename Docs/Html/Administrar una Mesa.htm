ADMINISTRAR UNA MESA

 

Para que aparezca las opciones que nos brinda la mesa, solo tiene que presionar clic derecho sobre la mesa, las opciones que se observan son:

  • Administrar mesa\Reservar: Esta opción se encarga de reservar la mesa, el estado es gráfico para identificar que mesas están disponibles ó no.
  • Administrar mesa\Ocupar: Esta opción se encarga de ocupar la mesa, el programa le preguntará si la desea atender inmediatamente.
  • Administrar mesa\Atender: Esta opción es la importante ya que es la operación de atender el punto de venta.
  • Administrar mesa\Cancelar: Esta opción se encarga de disponer de la mesa, solo si no tiene pedidos realizados.

  • Acciones\Nombre de la Mesa: Esta opción se encarga de personalizar el nombre de la mesa por parte del usuario.
  • Acciones\Pasar de Mesa: Pasa la información de una mesa a otra.
  • Acciones\Unir Mesas: Esta opción se encarga unir varias mesas a una sola mesa.
