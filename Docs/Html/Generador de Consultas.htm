GENERADOR DE CONSULTAS

Esta opción se encarga de hacer consultas personalizadas de los diferentes tipos de consultas y ofrece los siguientes campos así:

Datos Básicos

  • Código: Almacena el código del generador de consultas.
  • Nombre: Almacena el nombre ó descripción del generador de consultas.
  • Submenu: Almacena el nombre del submenu que aparecerá en el menu de consultas.
  • Opción:Almacena el nombre de la opción que aparecerá en el menu de consultas.
  • Ruta Informe: Almacena la ruta del informe prediseñado.

 

Información Tabla(s)

  • Tabla Maestra: Este campo contendrá la información de la tabla maestra de donde se tomará la información.
  • Tabla Detalle: Este campo contendrá la información de la tabla detalle de donde se tomará la información.
  • Link Campo Tabla Maestra: Este campo contendrá la información del campo que hará referencia a la tabla maestra con respecto a la tabla detalle.
  • Link Campo Tabla Detalle:Este campo contendrá la información del campo que hará referencia a la tabla detalle con respecto a la tabla maestra.

 

Sentencia SQL

    • En este campo se arma la sentencia SQL de donde saldrá la información, en caso de no tener un pleno conocimiento acerca de SQL solo basta con presionar el boton que dice "Generar SQL Base", inmediatamente el programa generará el código fuente SQL. Si en lagún momento desea agregar otra tabla u otro campo solo presione el botón de Agregar Tabla ó Agregar Campo.
    • También posee un botón que hace la verificación de la sentencia SQL que se ha digitado ó generado.
    • El botón "Distribuir Campo" se encarga de distribuir los campos por defecto al campo de "Campos Tipo Página" y al campo de "Campos Tipo Valores".
    • Nota: Al menos debe existir un campo de "Tipo Valor" para que la generación de la consulta sea exitosa.

 

Campos Tipo Página

Almacena los campos que estará en la consulta que estará disponible para ser luego pivoteados.

 

Campos Tipo Filas

Almacena los campos que estará en la consulta que estará en las filas por defecto.

Campos de Tipos Columnas

Almacena los campos que estará en la consulta que estará en las columnas por defecto.

 

Campos de Tipos Valores

Almacena los campos que estará en la consulta que estará en los valores por defecto.

 

Campos Para Pedir Rangos

En este campo se ofrecerá los campo para pedir rango de datos al momento de realizar dicha consulta.
