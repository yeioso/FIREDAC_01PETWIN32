INSTALACION

Antes de realizar este proceso, deberá estar creada en alguna base base de datos

La bases de datos que soporta la aplicación son:

  • MS-SQL SERVER
  • ORACLE
  • MS-ACCESS 
  • FIREBIRD

El sistema le ofrecerá una pantalla de configuración, la cual usted como usuario podrá seleccionar el tipo de base de datos que manejará su información.
